Assumption screening is part of the workflow

Each test now audits the assumptions it depends on before you interpret the p-value. These are data-driven screening checks, not proof of validity.

When assumptions look weak, the lab flags the risk and recommends a safer fallback such as Mann-Whitney, Welch ANOVA, Kruskal-Wallis, Spearman correlation, transformation, or category merging.

Independent Samples T-Test (Welch's)

Tests whether the means of a numeric variable differ significantly between two independent groups. Welch's method handles unequal variances.

H₀: μ₁ = μ₂  |  H₁: μ₁ ≠ μ₂
t = (x̄₁ − x̄₂) / √(s₁²/n₁ + s₂²/n₂)  ·  Welch-Satterthwaite df
Effect Size (Cohen's d)

One-Way Analysis of Variance (ANOVA)

Compares means across three or more groups simultaneously. Tests whether at least one group mean differs from the others.

H₀: μ₁ = μ₂ = ... = μₖ  |  H₁: at least one μᵢ differs
F = MS_between / MS_within  ·  df₁ = k−1, df₂ = N−k
Effect Size (η²)

Pearson Correlation Analysis

Measures the linear relationship between two continuous variables. Produces the correlation coefficient r ∈ [−1, 1] with a significance test.

r = Σ(xᵢ−x̄)(yᵢ−ȳ) / √[Σ(xᵢ−x̄)² · Σ(yᵢ−ȳ)²]
t = r√(n−2) / √(1−r²)  ·  df = n−2
Correlation Strength
Select two numeric variables and click Run Test.

Simple Linear Regression

Fits a linear model ŷ = β₀ + β₁x to the data. Tests whether the predictor explains significant variance in the outcome.

ŷ = β₀ + β₁x  ·  β₁ = Σ(xᵢ−x̄)(yᵢ−ȳ) / Σ(xᵢ−x̄)²
R² = 1 − SS_res/SS_tot  ·  t = β₁/SE(β₁), df = n−2
Select predictor and outcome variables and click Fit Model.

Chi-Square Test of Independence

Tests whether two categorical variables are statistically independent. A significant result suggests an association between the variables.

χ² = Σ (Oᵢⱼ − Eᵢⱼ)² / Eᵢⱼ  ·  Eᵢⱼ = (rowᵢ × colⱼ) / N
df = (r−1)(c−1)  ·  Cramér's V = √(χ²/N·min(r−1, c−1))
Choose two categorical variables and click Run Test.
